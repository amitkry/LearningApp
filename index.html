HTML stands for HyperText Markup Language. It is used to create the structure of web pages.

CSS stands for Cascading Style Sheets. It is used to style and design the HTML elements (colors, fonts, layouts).

JavaScript is a programming language used to add interactivity and dynamic behavior to web pages.

HTML gives the structure of a webpage, while CSS is used to make it look attractive.

Tags are special keywords enclosed in <> that tell the browser how to display content. Example: <p> for paragraph.
